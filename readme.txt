Use data from Advanced Custom Fields in W4 Post List templates.

=== Description ===

This plugin adds an [acf] shortcode to W4 Post List. ACF filed data can be accessed using [acf fieldname] or [acf field=fieldname]. Grouped/nested ACF fields can be accessed via dot notation: [acf group.subfield].
